#### Resolution dependent LESS rules

Compiles to a set of css files in the application directory `./build/css/*.css`.

Global variables:

    // global absolute paths
    @pathApp
    @pathImgFull

    // relative paths
    @pathImg

    // window dimension
    @height
    @width

    // safe zone padding
    @availTop
    @availRight
    @availBottom
    @availLeft

    // font sizes
    @fontSize

    // visible page content size
    @availHeight
    @availWidth
