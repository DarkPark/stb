STB framework command-line tools
================================

## Getting Started

Install `stb-cli` globally:

```bash
npm install -g stb-cli
```

## Usage

Create a new folder for your project:

```bash
mkdir myapp
```

Run the generator from within the new folder:

```bash
cd myapp
stb-cli init
```
