#### Resolution dependent LESS rules

Compiles to a set of css files in the application directory `./build/develop/css/*.css` and `./build/release/css/*.css`.

Global variables:

	// window dimension
	@height
	@width

	// safe zone padding
	@availTop
	@availRight
	@availBottom
	@availLeft

	// font sizes
	@fontSize

	// visible page content size
	@availHeight
	@availWidth
